# 翱翔教务功能加强 

基于 [翱翔教务功能加强](https://greasyfork.org/zh-CN/scripts/524099-%E7%BF%B1%E7%BF%94%E6%95%99%E5%8A%A1%E5%8A%9F%E8%83%BD%E5%8A%A0%E5%BC%BA) 油猴脚本 v1.7.2 二次开发，**非原作者**。

## 新增功能

- **GPA 预测**：为当前学期未出分课程设定预估绩点，计算预测总 GPA。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)。
2. 新建脚本，将 `main.user.js` 的内容粘贴进去保存即可。
